# SpiderStack

A production-ready starter kit for building full-stack web applications with [Spider](https://www.spiderme.org/) — the Zig web framework.

**Live demo**: [https://www.spiderme.org/](https://www.spiderme.org/)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Language | Zig `0.17.0-dev` |
| Framework | [Spider](https://www.spiderme.org/) |
| Database | PostgreSQL 16 |
| Auth | JWT + Google OAuth2 |
| HTTP Client | [pacman](https://github.com/llllOllOOll/pacman) |
| CSS | Tailwind CSS + DaisyUI |
| Frontend | HTMX + Alpine.js |
| PWA | Service Worker + Web Manifest |

---

## Features

- **Google OAuth2** — one-click login via Google
- **Email/password auth** — register and login with email
- **JWT sessions** — secure HttpOnly cookies
- **RBAC** — role-based access control
- **HTMX** — seamless page updates without full reload
- **i18n** — Portuguese (pt-BR) and English (en-US)
- **PWA** — installable, offline-capable
- **CRUD** — Games and Todo list with full create/read/update/delete
- **Migrations** — automated database schema management
- **Embedded templates** — compiled into the binary, zero runtime file reads

## Getting Started

### 1. Prerequisites

**Zig** `0.17.0-dev`:
```bash
zig version
# 0.17.0-dev.93+76174e1bc
```

**PostgreSQL client library**:

```bash
# Arch Linux
sudo pacman -S postgresql-libs

# Ubuntu/Debian
sudo apt install libpq-dev

# Fedora
sudo dnf install postgresql-devel

# macOS
brew install libpq
```

### 2. Start PostgreSQL

```bash
docker-compose up -d
```

### 3. Configure environment

```bash
cp .env.example .env
# edit .env with your values
```

### 4. Build and run

```bash
zig build run
```

Open `http://127.0.0.1:8080`

### 5. Build CSS (optional — only if editing styles)

```bash
npm install
npm run build:css

# or watch mode during development
npm run watch:css
```

## How Spider is Used

### Server setup

```zig
// main.zig

// One line — Spider detects templates automatically at compile time
pub const spider_templates = @import("embedded_templates.zig").EmbeddedTemplates;

pub fn main(init: std.process.Init) !void {
    try spider.pg.init(init.arena.allocator(), init.io, .{
        .host = spider.env.getOr("POSTGRES_HOST", "localhost"),
        .port = spider.env.getInt(u16, "POSTGRES_PORT", 5432),
        .user = spider.env.getOr("POSTGRES_USER", "spider"),
        .password = spider.env.getOr("POSTGRES_PASSWORD", ""),
        .database = spider.env.getOr("POSTGRES_DB", "spiderdb"),
    });
    defer spider.pg.deinit();

    var server = spider.app();
    defer server.deinit();

    server
        .get("/login", auth.controller.index)
        .get("/auth/google", auth.controller.redirectToGoogle)
        .get("/auth/google/callback", auth.controller.googleCallback)
        .use(middleware.auth)
        .get("/", home.controller.index)
        .get("/games", games.controller.index)
        .listen(8080) catch |err| return err;
}
```

### Handler pattern

```zig
// features/home/controller.zig

pub fn index(c: *spider.Ctx) !spider.Response {
    const locale = i18n.localeFromHeader(c.header("Accept-Language"));
    const context = try presenter.buildHomeContext(c.arena, c, locale);
    return c.view("home/index", context, .{});
}
```

### Database queries

```zig
// features/games/repository.zig

pub fn findAll(alloc: std.mem.Allocator) ![]Game {
    return spider.pg.query(Game, alloc,
        "SELECT id, name, platform, release_year FROM games ORDER BY name",
        .{},
    );
}
```

### Auth middleware

```zig
// core/middleware/auth_middleware.zig

var gAuth = spider.auth.Auth.init(.{
    .secret = spider.env.getOr("JWT_SECRET", "changeme"),
    .public_paths = &.{
        "/login",
        "/auth/google",
        "/auth/google/callback",
        "/auth/email/register",
        "/auth/email/login",
    },
    .redirect_to = "/login",
    .secure_cookie = false,
});

pub fn auth(c: *spider.Ctx, next: spider.NextFn) !spider.Response {
    return gAuth.middleware(c, next);
}
```

### HTMX partial rendering

```zig
// features/todo/controller.zig

pub fn index(c: *spider.Ctx) !spider.Response {
    const todos = try repository.findAll(c.arena);
    // Spider automatically returns only the content block for HTMX requests
    return c.view("todo/index", .{ .todos = todos }, .{});
}
```

## Data Model

### users
```sql
id          SERIAL PRIMARY KEY
uuid        UUID UNIQUE NOT NULL
email       VARCHAR(255) UNIQUE NOT NULL
name        VARCHAR(255)
locale      VARCHAR(10) DEFAULT 'pt-BR'
locale_set  BOOLEAN DEFAULT FALSE
created_at  TIMESTAMPTZ DEFAULT NOW()
```

### games
```sql
id              SERIAL PRIMARY KEY
name            VARCHAR(255) NOT NULL
platform        VARCHAR(100)
release_year    INTEGER
genre           VARCHAR(100)
developer       VARCHAR(255)
sales_millions  DECIMAL(10,2)
rating          DECIMAL(3,1)
created_at      TIMESTAMPTZ DEFAULT NOW()
```

### todos
```sql
id          SERIAL PRIMARY KEY
title       VARCHAR(255) NOT NULL
completed   BOOLEAN DEFAULT FALSE
created_at  TIMESTAMPTZ DEFAULT NOW()
updated_at  TIMESTAMPTZ DEFAULT NOW()
```
